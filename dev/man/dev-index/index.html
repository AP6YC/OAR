    OAR.BNF — Type
    struct BNF <: OAR.Grammar

    Summary

    Backus-Naur form of Grammar.

    Consists of a set of terminal symbols, non-terminal symbols, and production rules.

    Fields

    • N::Set{OAR.GSymbol}: Non-terminal symbols of the grammar.
    • T::Set{OAR.GSymbol}: Terminal symbols of the grammar.
    • S::Set{OAR.GSymbol}: Definition of a statement in this grammar.
    • P::Dict{OAR.GSymbol, Set{OAR.GSymbol}}: The set of production rules of the grammar.
